01

Foundations

Recursive Frontier

Every Affine surface is assembled from the same primitives. A table row is never just a row — it is a candidate, a memory, a winner, a failed path or an inherited state. If a visual element can't be mapped to a primitive, it is decorative.

Candidate

Independent capabilities entering — rows, traces, challenger marks.

Pressure

Shared constraints that make work comparable — axes, rubrics, thresholds.

Reinforcement

Useful paths strengthening — heavier lines, improving series.

Recession

Weak paths failing — faded rows, failed traces, low-alpha history.

Resolution

One path becoming clearer — champion marker, selected threshold.

Inheritance

Validated progress becoming baseline — gold trace carried forward.

Memory

Prior eras remaining operationally relevant — strata, residue.

Reopening

New possibility from inherited state — challenger queue, empty slots.

Diverge
Align

The operating cycle — every mode is a different rendering of this sequence

Motion grammar

Consequence, not ambience

Motion reveals cause, consequence and accumulated change. Mechanical qualities (stable intervals, thresholds, channels) over living qualities (local variation, reinforcement, weak paths fading). Never a full-viewport gold flash on load.

Protocol frame

The rules are the intelligence.

Affine does not ask miners to share intentions. It makes intention less important than consequence. Local attempts become globally useful only when they survive the same public replacement rule.

The mechanism is severe by design: one committed pointer, one scheduled challenge, one champion state, one inherited frontier.

Validator role

Host the field, not the frontier.

Validators run committed revisions through the shared environments and set weights toward the champion. Miners improve models; validators make the contest reproducible.

validator command public evaluation

# run the shared evaluation field
